# 3D Anatomy Model - Lista de Meshes

Este documento lista todos os 368 meshes disponíveis no modelo 3D de anatomia humana (`anatomy-internal.glb`).

## Visão Geral

O modelo permite alternar entre duas visualizações:
- **Externa (external)**: Mostra corpo, olhos, sobrancelhas e cílios
- **Interna (internal)**: Oculta a pele para revelar músculos, esqueleto, órgãos e sistemas internos

## Uso no Código

### Alternar Visibilidade por Nome

```typescript
// Exemplo: ocultar/mostrar mesh específico
clonedScene.traverse((child) => {
  if (child instanceof THREE.Mesh) {
    if (child.name === 'heart') {
      child.visible = showHeart;
    }
  }
});
```

### Filtrar por Categoria

```typescript
// Exemplo: mostrar apenas sistema esquelético
const skeletalMeshes = [
  'skull', 'jaw_bone', 'cervical_spine', 'thoracic_spine',
  'lumbar_spine', 'sacrum', 'coccyx', 'thorax', 'sternum',
  // ... adicione todos os ossos
];

clonedScene.traverse((child) => {
  if (child instanceof THREE.Mesh) {
    child.visible = skeletalMeshes.includes(child.name);
  }
});
```
